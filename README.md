# 🌐 Desenvolvimento Fullstack com Django

Este projeto tem como **objetivo geral** fornecer uma compreensão abrangente e prática da criação de aplicativos web utilizando o poderoso framework **Django**.  
A proposta é dominar os conceitos fundamentais e aplicar as melhores práticas para construir aplicações robustas e escaláveis.

## ⚙️ Instalação e Configuração

Clone o repositório:

- git clone https://github.com/PatriciaGheller/desenvolvimento_fullstack_com_django.git

- cd desenvolvimento_fullstack_com_django

- Instale as dependências com Poetry:

poetry install

- Ative o ambiente virtual:

poetry shell

- Verifique a versão do Django instalada:

poetry run python -m django --version

## 🛠️ Executando o Projeto

- Crie o projeto Django:

poetry run django-admin startproject config .

- Inicie o servidor local:

poetry run python manage.py runserver

- Acesse no navegador: http://127.0.0.1:8000/ (127.0.0.1 in Bing)
